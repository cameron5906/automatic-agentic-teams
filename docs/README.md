## Contributing

### Adding an ADR

1. Create a new file in `docs/adr/` with format: `YYYY-MM-DD-short-description.md`
2. Use the ADR template from `docs/adr/README.md`
3. Update this README's ADR table
4. Update `docs/adr/README.md` table of contents

### Adding Research

1. Create a new file in `docs/research/` with format: `YYYY-MM-DD-short-description.md`
2. Use the research template from `docs/research/README.md`
3. Update this README's Research table
4. Update `docs/research/README.md` table of contents
